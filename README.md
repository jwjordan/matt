# Getting Started
## Install dependencies
```
npm install
```

## Run app locally
```
npm start
```

## Compile native app
```
npm i -g @electron-forge/cli;
electron-forge make
```

## Publish a version
1. Make the packages
```
electron-forge make
```
1. Create a release in GitHub web UI
1. Upload the release package binaries (.zip for MacOS, .exe for Windows) to the release
